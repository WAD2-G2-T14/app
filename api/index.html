Supreme Leader's Shopping API Documentation

Authentication

POST /api/login.php

This api handles user logins and returns the user id and details if successfully verified.

Payload
  • username*
  • password*
Response Example
  • "name": "Jonathan Tan Jia Wei"
  • "phone": "91234567"
  • "birthday": "1998-12-25"
  • "user_id": "5fa40d29e57d4"
  • "email": "jonathan@mail.com"

POST /api/signup.php

This api handles user signups.

Payload
  • email*
  • password*
  • name*
  • postal_code*
  • address*
  • city*
  • country*
  • phone_number*

Items

GET /api/items/all.php

This api returns the expiry and order status for all items

Payload
  • none
Response Example
  • "01351492" :
    • "current_orders" : 5
    • "total_required" : 10
    • "expiry" : "2020-12-25"

GET /api/items/show_one.php

This api returns the expiry and order status for one item

URL Params
  • item_id=01351492
Response Example
  • "current_orders" : 5
  • "total_required" : 10
  • "expiry" : "2020-12-25"

PUT /api/items/order.php

This api handles the ordering logic and accepts the user's cart details

Payload
  • "user_id": "01351492"
  • "item_list": (array)
    • "0" =>
    • "item_id": "01351491"
    • "quantity" "1"
    • "1" =>
    • "item_id": "01351492"
    • "quantity": "2"
    • ...
